You can scroll

~Exploration~
---------------
This screen aims
to help with
exploration. It
shows terse data
about the last
scanned object.

Swap between
scans in the
current system
by pressing on
the right scroll
wheel.

Stars of types
BAFGKM provides
an estimated
Habitable Zone
range. This can
be used as an
guideline for
finding
terraformable
planets.

~Materials~
----------------
This screen
keeps track of
changes to
materials,
ordered by most
recently changed
first. Press the
right wheel to
reset the
counts.
